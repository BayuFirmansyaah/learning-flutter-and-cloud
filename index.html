Cara Menggunakan Materi Ini

  1. Buka setiap halaman secara berurutan
  2. Ikuti setiap langkah praktik dengan teliti
  3. Jalankan setiap command yang diberikan di terminal Anda
  4. Verifikasi hasil setiap langkah sebelum lanjut
  5. Jika ada error, baca pesan error dengan seksama

Persyaratan Sistem

  • OS: Windows 10/11 (64-bit)
  • RAM: Minimal 8GB (rekomendasi 16GB)
  • Storage: Minimal 10GB ruang kosong
  • Koneksi Internet: Stabil untuk download tools & deploy
  • Browser: Chrome atau Edge versi terbaru

Tools yang Akan Digunakan

  • Flutter SDK
  • Android Studio atau VS Code
  • Git (untuk version control)
  • Google Cloud CLI (gcloud)
  • Docker (opsional, untuk Cloud Run)
