Panduan Praktis Flutter untuk Windows

Bagian 1: Instalasi Flutter SDK

Langkah 1: Download Flutter SDK

  1. Buka browser dan kunjungi: 
    https://flutter.dev/docs/get-started/install/windows
  2. Download file ZIP Flutter SDK versi stabil terbaru
  3. Ukuran file sekitar 1-2 GB, pastikan koneksi internet stabil

Langkah 2: Ekstrak Flutter SDK

  1. Buat folder di lokasi yang mudah diakses (jangan di Program Files): 
    C:\src\flutter
  2. Ekstrak file ZIP yang sudah didownload ke folder tersebut
  3. Pastikan struktur folder menjadi: C:\src\flutter\bin

Langkah 3: Tambahkan Flutter ke PATH

  1. Buka Settings Windows
  2. Search "Environment Variables" atau "Edit system environment variables"
  3. Klik tombol Environment Variables
  4. Di bagian User variables, cari variabel Path
  5. Klik Edit, lalu New
  6. Tambahkan: 
    C:\src\flutter\bin
  7. Klik OK pada semua dialog

Langkah 4: Verifikasi Instalasi Flutter

  1. Buka Command Prompt atau PowerShell
  2. Jalankan command: 
    flutter --version
  3. Output akan menampilkan versi Flutter yang terinstall
  4. Jalankan Flutter Doctor untuk cek kebutuhan lain: 
    flutter doctor
Catatan: Flutter Doctor akan menampilkan checklist tools yang diperlukan. Jangan khawatir jika ada tanda [X] atau [!], kita akan setup satu per satu.

Bagian 2: Setup Editor (VS Code)

Langkah 1: Install Visual Studio Code

  1. Download VS Code dari: 
    https://code.visualstudio.com/
  2. Install dengan settingan default
  3. Buka VS Code setelah instalasi selesai

Langkah 2: Install Extension Flutter & Dart

  1. Buka VS Code
  2. Klik icon Extensions di sidebar kiri (atau tekan Ctrl+Shift+X)
  3. Search "Flutter" dan install extension resmi dari Dart Code
  4. Extension Dart akan otomatis terinstall juga
  5. Restart VS Code setelah instalasi selesai

Bagian 3: Setup Android Development (Opsional tapi Direkomendasikan)

Langkah 1: Install Android Studio

  1. Download Android Studio dari: 
    https://developer.android.com/studio
  2. Install dengan settingan default
  3. Saat pertama buka, ikuti wizard setup untuk download Android SDK
  4. Proses ini membutuhkan waktu karena download components (~3-5 GB)

Langkah 2: Setup Android SDK

  1. Buka Android Studio
  2. Pilih More Actions → SDK Manager
  3. Di tab SDK Platforms, pastikan ada minimal satu Android version tercentang
  4. Di tab SDK Tools, pastikan tercentang:
    • Android SDK Build-Tools
    • Android SDK Command-line Tools
    • Android Emulator
    • Android SDK Platform-Tools
  5. Klik Apply untuk install

Langkah 3: Accept Android Licenses

  1. Buka Command Prompt atau PowerShell
  2. Jalankan: 
    flutter doctor --android-licenses
  3. Ketik y untuk menerima semua licenses

Langkah 4: Verifikasi Setup Android

  1. Jalankan lagi: 
    flutter doctor
  2. Pastikan baris Android toolchain menunjukkan tanda [✓]

Bagian 4: Membuat Proyek Flutter Pertama

Langkah 1: Buat Proyek Baru via Command Line

  1. Buka Command Prompt atau PowerShell
  2. Navigasi ke folder tempat Anda ingin membuat proyek: 
    cd C:\Users\NamaAnda\Documents
  3. Buat proyek Flutter baru: 
    flutter create my_first_app
  4. Tunggu proses selesai (1-2 menit)
  5. Masuk ke folder proyek: 
    cd my_first_app

Langkah 2: Buka Proyek di VS Code

  1. Jalankan command: 
    code .
  2. Atau buka VS Code dan pilih File → Open Folder → pilih folder my_first_app

Langkah 3: Lihat Struktur Proyek

Proyek Flutter memiliki struktur sebagai berikut:

my_first_app/
├── android/          # Konfigurasi Android
├── ios/              # Konfigurasi iOS
├── lib/              # Kode Dart Anda (folder utama)
│   └── main.dart     # File entry point aplikasi
├── test/             # Unit tests
├── pubspec.yaml      # Dependencies & metadata proyek
└── README.md         # Dokumentasi proyek
Penting: Semua kode aplikasi Anda akan ada di folder lib/, terutama di file main.dart

Bagian 5: Menjalankan Aplikasi Flutter

Opsi 1: Jalankan di Chrome (Paling Mudah)

  1. Pastikan Chrome sudah terinstall
  2. Di VS Code, tekan F5 atau pilih Run → Start Debugging
  3. Pilih Chrome sebagai device
  4. Tunggu build process (pertama kali agak lama, ~2-3 menit)
  5. Aplikasi akan terbuka di browser Chrome

Opsi 2: Jalankan via Command Line

  1. Buka terminal di VS Code (Ctrl+`)
  2. Jalankan: 
    flutter run -d chrome
  3. Tunggu hingga aplikasi terbuka di browser

Opsi 3: Jalankan di Android Emulator (Jika Sudah Setup)

  1. Buka Android Studio → Device Manager
  2. Buat virtual device baru atau jalankan yang sudah ada
  3. Di VS Code, pilih device Android dari dropdown di pojok kanan bawah
  4. Tekan F5 untuk run
  5. Atau via terminal: 
    flutter run

Bagian 6: Memahami Kode Aplikasi Pertama

File main.dart

Buka lib/main.dart. Ini adalah struktur dasar aplikasi Flutter:

import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),
        useMaterial3: true,
      ),
      home: const MyHomePage(title: 'Flutter Demo Home Page'),
    );
  }
}

Penjelasan singkat:

  • main() - Entry point aplikasi
  • runApp() - Menjalankan widget root
  • MyApp - Widget root aplikasi
  • MaterialApp - Wrapper untuk Material Design
  • MyHomePage - Halaman utama aplikasi (stateful widget)

Bagian 7: Modifikasi Aplikasi Pertama

Latihan 1: Ubah Teks di Home Page

  1. Cari baris yang berisi You have pushed the button this many times:
  2. Ubah teks tersebut menjadi: 
    'Aplikasi Flutter Pertama Saya:'
  3. Simpan file (Ctrl+S)
  4. Aplikasi akan otomatis reload (hot reload)

Latihan 2: Ubah Warna Tema

  1. Di class MyApp, cari baris: 
    colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),
  2. Ubah Colors.deepPurple menjadi Colors.blue atau warna lain
  3. Simpan dan lihat perubahan warna aplikasi

Latihan 3: Ubah Judul App Bar

  1. Cari baris: 
    home: const MyHomePage(title: 'Flutter Demo Home Page'),
  2. Ubah teks title menjadi nama Anda atau judul lain
  3. Simpan dan cek perubahan di app bar

Bagian 8: Hot Reload vs Hot Restart

Hot Reload

  • Otomatis terjadi saat save file (Ctrl+S)
  • Atau tekan r di terminal
  • Mempertahankan state aplikasi
  • Sangat cepat (~1 detik)
  • Cocok untuk perubahan UI

Hot Restart

  • Tekan Ctrl+Shift+F5 di VS Code
  • Atau tekan R di terminal
  • Mereset state aplikasi ke awal
  • Lebih lambat dari hot reload (~3-5 detik)
  • Gunakan saat mengubah kode yang mempengaruhi state atau main()

Bagian 9: Debugging Flutter

Menggunakan Print Statements

  1. Tambahkan di kode Anda: 
    print('Debug: nilai counter = $_counter');
  2. Output akan muncul di Debug Console VS Code

Breakpoints

  1. Klik di sebelah kiri nomor baris untuk set breakpoint (titik merah muncul)
  2. Run aplikasi dalam debug mode (F5)
  3. Eksekusi akan berhenti di breakpoint
  4. Inspect variabel di panel Variables

Flutter DevTools

  1. Saat aplikasi running, buka link DevTools yang muncul di terminal
  2. Atau jalankan: 
    flutter pub global activate devtools
flutter pub global run devtools
  3. DevTools menyediakan inspector, performance monitoring, dan memory profiler

Bagian 10: Command-Command Penting Flutter

Informasi & Diagnostik

flutter doctor             # Cek setup environment
flutter doctor -v          # Versi verbose
flutter --version          # Cek versi Flutter
flutter devices            # List devices yang tersedia

Manajemen Proyek

flutter create nama_app    # Buat proyek baru
flutter pub get            # Download dependencies
flutter pub upgrade        # Update dependencies
flutter clean              # Bersihkan build files

Menjalankan Aplikasi

flutter run                # Run di device default
flutter run -d chrome      # Run di Chrome
flutter run -d windows     # Run di Windows desktop
flutter run --release      # Run dalam release mode

Build Aplikasi

flutter build apk          # Build APK untuk Android
flutter build appbundle    # Build App Bundle untuk Play Store
flutter build web          # Build untuk web
flutter build windows      # Build untuk Windows desktop

Testing & Analisis

flutter test               # Jalankan unit tests
flutter analyze            # Analisis kode untuk issues
flutter format lib/        # Format kode otomatis

Checklist: Anda Berhasil Jika...

  • ✓ flutter doctor menampilkan minimal 1 device ready
  • ✓ Anda bisa membuat proyek baru dengan flutter create
  • ✓ Aplikasi bisa dijalankan di Chrome atau emulator
  • ✓ Hot reload bekerja saat Anda save file
  • ✓ Anda bisa mengubah teks dan warna aplikasi
  • ✓ Anda memahami struktur folder lib/ dan main.dart

Troubleshooting Umum

Error: 'flutter' is not recognized

Solusi: PATH belum di-set dengan benar. Ulangi Langkah 3 di Bagian 1, lalu restart Command Prompt.

Error: Android licenses not accepted

Solusi: Jalankan flutter doctor --android-licenses dan ketik y untuk semua.

Error: Unable to locate SDK

Solusi: Install Android Studio dan jalankan setup wizard untuk download Android SDK.

Build gagal dengan error Gradle

Solusi: Jalankan flutter clean lalu flutter pub get, kemudian build ulang.

Hot reload tidak bekerja

Solusi: Gunakan hot restart (Ctrl+Shift+F5) atau restart aplikasi sepenuhnya.
